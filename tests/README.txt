================ TESTS LIBRARY CCSD


Ces Tests concerne les classes du projet Library du CCSD.

==== Dépendance
Ces tests utilisent PHPUnit

==== Hiérarchie des fichiers
Reprend la hiérarchie des fichiers du projet, conformément à l'utilisation de PHPUnit.

=== Utilisation
Test simple : phpunit --bootstrap tests/bootstrap.php tests/Ccsd/Meta/BibcodeTest.php
Tests complets : phpunit --bootstrap tests/bootstrap.php tests/
